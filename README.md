# Rhine
 Rhine is an actual rhythm game inspired by Dancing Line, has similar but different mechanics than Dancing Ballz.

## Control
 Use your 4 arrow keys ⬆️⬅️⬇️➡️ on your keyboard to control the line.

## Judgement
 🟥 Bad/Miss 🟨 Perfect 🟦 Good
